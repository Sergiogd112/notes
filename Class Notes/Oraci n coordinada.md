# Tipología

- Yuxtapuestas

    Nexos:

    - ,
    - :
    - ;
- Copulativas

    Indican unión o suma

    Nexos:

    - y
    - e
    - ni
- Adversativas

    Expresan oposición o contraste.

    Nexos:

    - pero
    - sino (que)
    - sin embargo
    - ahora bien
    - no obstante
- Disyuntivas

    Indican significados que se excluyen entre sí

    Nexos:

    - o
    - u
    - o bien ... o bien
- Distributiva

    Indican una referencia o elección alternativa

    Nexos:

    - bien...bien
    - ya...ya
    - unos...otros
    - tan pronto... como...
- Explicativa

    Una proposición explica o aclara el significado de la anterior.

    Nexos:

    - es decir
    - esto es
    - o sea
