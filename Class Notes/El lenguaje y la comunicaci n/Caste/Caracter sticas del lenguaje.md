# Características del lenguaje

Created: Oct 09, 2019 6:27 PM

# Discreción

Esta formado por unidades minimas discretas. Pueden: 

- Tener significado⇒morfemas
- No tener significados⇒fonemas

# Dualidad de patron

Las lenguas naturales combinan elementos min sin significado para formar elementos con significado.

# Infinidad discreta

Capacidad de unir elementos finitos y generar infinitos

Inefable: concepto que se utiliza para referirse a aquello que no puede explicarse con palabras. Lo inefable, por lo tanto, no puede narrarse o expresarse.

# Recursividad
